# Heavenly Reviews Collector

特定のレビューサイトからデータを収集・管理するChrome拡張機能。

## 使い方

### 1. 準備

収集したいサイトの **レビュー一覧（1ページ目）** を開き、**URLをコピー** します。

### 2. 収集開始

1. 拡張機能アイコン → **「オプションを開く」** をクリック。
2. 管理画面へ移動後、入力欄に **URLを貼り付け**。
3. **「収集開始」** をクリック。

### 3. 収集中の動作
「Collecting...」と表示されれば正常に動いています。

バックグラウンドで処理が進むため、完了までブラウザを閉じないでください。

### 4. データ管理
収集完了後、ダッシュボードへ自動遷移します。

*   **確認**: テーブルでデータを閲覧。
*   **保存**: 右上のボタンから CSV / TSV / JSON でエクスポート可能。
*   **リセット**: 「クリア」ですべてのデータを削除。
